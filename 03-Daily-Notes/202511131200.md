## IDEMPOTÊNCIA

Faz uma ação X vezes e o resultado ser o mesmo, nao ser alterado
Exemplo de um post e voce mandar um post X vezes ele sempre vai ser o mesmo, sempre vai criar um novo com as informações
Exemplo de algo nao idempodente seria o put, pois voce sempre esta alterando o conteudo e caso voce alterde sempre por cima ele nao vai ser o mesmo
Outro exemplo e de voce clicar para fazer uma compra 3x e ele efetivar as 3 compas no seu cartao
IDEMPOTÊNCIA KEY 
Pode existir uma idempodency key para salvar uma transação com X valores e dai ele sempre vai intepretar esses X valores como apenas uma transição, exemplo REAL, 100,PIX,13/11/2025 22:48, todos que forem com os mesmo valores vao ser interpretadas como uma.
